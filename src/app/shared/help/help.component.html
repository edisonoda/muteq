Como leio um Código QR?

Ao abrir o Leitor de Código QR (botão amarelo na parte direita inferior), o sistema irá pedir permissão para abrir a câmera. Conceda a permissão.


Com a câmera aberta (no Leitor de Código QR), aponte para o Código QR do item na exposição do museu.

Pronto! O sistema irá levá-lo às informações do item.


O leitor pode demorar alguns segundos para iniciar.
Tenha certeza que a câmera está limpa e bem posicionada. ...
